## Short Courses (< 5 hours)

### 1. ChatGPT Prompt Engineering for Developers

- **Provider:** DeepLearning.AI + OpenAI
- **Instructor:** Isa Fulford, Andrew Ng
- **Level:** 🟢
- **Duration:** ~1.5 hours
- **Cost:** Free
- **Link:** [deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/](https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/)

**What You Will Learn:**
- Prompting best practices for software development
- How to use LLM APIs for summarization, inference, transformation, and expansion
- Building a custom chatbot with system prompts
- Understanding tokenization and context windows

**Prerequisites:** Basic Python knowledge.

---

### 2. Building Systems with the ChatGPT API

- **Provider:** DeepLearning.AI + OpenAI
- **Instructor:** Isa Fulford, Andrew Ng

- **Duration:** ~1.5 hours

- **Link:** [deeplearning.ai/short-courses/building-systems-with-chatgpt/](https://www.deeplearning.ai/short-courses/building-systems-with-chatgpt/)

### 4. Building and Evaluating Advanced RAG Applications

- **Provider:** DeepLearning.AI + LlamaIndex
- **Instructor:** Jerry Liu, Andrew Ng

- **Link:** [deeplearning.ai/short-courses/building-evaluating-advanced-rag/](https://www.deeplearning.ai/short-courses/building-evaluating-advanced-rag/)

### 9. Stanford CS324: Large Language Models

- **Provider:** Stanford University
- **Instructor:** Percy Liang, Tatsunori Hashimoto
- **Level:** 🔴
- **Duration:** ~40 hours (full quarter)
- **Cost:** Free (materials online)
- **Link:** [stanford-cs324.github.io/winter2022/](https://stanford-cs324.github.io/winter2022/)

- Foundations of language model training and architecture
- Scaling laws and emergent abilities

## Suggested Learning Paths

### Path 1: Complete Beginner (0-3 months)

```
Month 1: Foundations
├── 1. Hugging Face NLP Course (Chapters 1-4) ............. 🟢
├── 2. ChatGPT Prompt Engineering for Developers .......... 🟢
└── 3. LLM University (Cohere) ............................ 🟢

Month 2: Building Applications
├── 4. LangChain for LLM Application Development .......... 🟡
├── 5. LangChain: Chat with Your Data ..................... 🟡
└── 6. Vector Databases: From Embeddings to Applications ... 🟡

Month 3: Advanced Techniques
├── 7. Building and Evaluating Advanced RAG ............... 🟡
├── 8. Building Systems with the ChatGPT API .............. 🟡
└── 9. Full Stack LLM Bootcamp ............................ 🟡
```

```
Week 1-2: Rapid Foundations
├── 1. ChatGPT Prompt Engineering for Developers .......... 🟢
└── 2. LangChain for LLM Application Development .......... 🟡

Week 3-4: RAG Deep Dive
├── 3. LangChain: Chat with Your Data ..................... 🟡
├── 4. Advanced Retrieval for AI with Chroma .............. 🟡
└── 5. Preprocessing Unstructured Data .................... 🟡

Week 5-6: Production Systems
├── 6. Building and Evaluating Advanced RAG ............... 🟡
├── 7. Building Systems with the ChatGPT API .............. 🟡
└── 8. Full Stack LLM Bootcamp ............................ 🟡
```

### Path 3: ML Engineer Going Deep (2-4 months)

```
Phase 1: Architecture (2-3 weeks)
├── 1. Stanford CS25: Transformers United ................. 🔴
└── 2. Hugging Face NLP Course (Advanced Chapters) ........ 🟡

├── 3. Building and Evaluating Advanced RAG ............... 🟡

Phase 3: Research & Specialization (4+ weeks)
├── 6. Stanford CS324: Large Language Models .............. 🔴

└── 8. Build a production context engineering system ...... 🔴
```
